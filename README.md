## Installation & Usage
1. **Clone this repository**
   ```bash
   git clone https://github.com/idangadish/JokeHub.git
   ```
2. **Navigate into the project folder**
   ```bash
   cd JokeHub
   ```
3. **Install dependencies**
   ```bash
   npm install
   ```
4. **Run the app**
   ```bash
   npm start
   ```
5. **Open the app in your browser**
   ```
   http://localhost:3000
   ```
